# Dossier Video

Placez votre vidéo dans ce dossier avec le nom `djshek-video.mp4`.

## Formats supportés :
- **MP4** (recommandé) : `djshek-video.mp4`
- **WebM** (optionnel) : `djshek-video.webm`

## Recommandations :
- Format horizontal (16:9 recommandé)
- Résolution recommandée : 1920x1080 (Full HD) ou 1280x720 (HD)
- Codec vidéo : H.264 (MP4) ou VP9 (WebM)
- Taille de fichier : optimisez votre vidéo pour le web (< 50 MB si possible)

La vidéo apparaîtra automatiquement sur le site entre la playlist audio et la biographie officielle.
